2. Forward Web Application for oil stations

Forward Web Application for oil stations

  • The Forward Web Application is a client-side system designed for oil station management, allowing station managers to efficiently manage various aspects of station operations and customer interactions. This system helps optimize business processes and improve operational efficiency.
    • Product and Price Management: Station managers can dynamically adjust the prices of products, including B2B pricing for wholesale transactions.
    • Shift Management: Enables managers to schedule employee shifts, ensuring smooth operational workflows.
    • Delivery Management: Tracks and manages the physical transfer of crude oil and refined petroleum products from the seller to the buyer.
    • Customer Role Management: Customizes user roles, permissions, and access levels for different types of customers.
    • Customer Management: Credit/Debit Management: Customers can have credit or debit accounts with a specified credit limit or debit top-up, allowing flexible payment options.
    • Promotions: Supports promotional offers, discounts, and loyalty programs to boost customer engagement and retention.
    • POS Configuration: Allows configuration of the POS system according to business requirements.
    • User Management: Manages user authentication and roles for secure access to the system.
  • Technologies Used:
    • Backend: Built with Golang, providing high performance and scalability for efficient system operation.
    • Frontend: Developed using GWT (Java) for creating a responsive and intuitive user interface.
    • Database: SQLite3
